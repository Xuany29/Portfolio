# Herbarium Universal
Herbarium Universal is a PHP-based plant and herbarium website that helps users explore plant species, read classification content, follow DIY herbarium tutorials, and submit or identify plant-related information.
In this group project, I developed the DIY Herbarium section, including the Tutorials, Tools, and Care pages. I also implemented PHP backend functions to handle and validate user input in the contribution form. In addition, I worked on database integration to retrieve data for the admin view, and I enabled administrators to edit and delete records through the website.
## 🛠️ Tech Stack
- HTML  
- CSS  
- PHP

# Space Shooter
This is a 2D space shooter game developed in C using the SDL2 library. The game enables real-time player controls, enemy attacking, boss battles, collision detection, and score tracking. It incorporates core game development concepts such as event handling, rendering and memory management. Besides, some additional features such as health system and high score storage also have been implemented.
## 🛠️ Tech Stack
- C
- SDL2
- SDL2_image
- SDL2_ttf

# Database
This Medication Manager is a system designed to help doctors manage patient prescriptions more efficiently. It aims to consolidate various activities involved in prescribing medications for the treatment of chronic diseases.
In this group project, I was responsible for designing the database structure and implementing enhancements in the Software Development major. The enhancements I contributed include an automatic reminder system for patients after a doctor prescribes medication, as well as an automatic update of history records when prescriptions are edited or deleted.
## 🛠️ Tech Stack
- SQL
- MySQL

# AI Classification System
This system is designed to classify traffic conditions and predict optimal routes to improve traffic management. It involves three deep learning techniques to analyze and classify incident severity. Based on the predicted severity, the system can estimate travel time between locations and recommend the most optimal path for users.
## 🛠️ Tech Stack
- Python
- Tensorflow
